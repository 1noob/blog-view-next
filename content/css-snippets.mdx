## Diagonal line pattern

```css
.pattern {
  --fill: #ddd;
  background-image: linear-gradient(
    45deg,
    var(--fill) 7.25%,
    transparent 0,
    transparent 50%,
    var(--fill) 0,
    var(--fill) 57.25%,
    transparent 0,
    transparent
  );
  background-size: 8px 8px;
}
```

## Dot pattern

```css
.pattern {
  --fill: #ddd;
  background-image: radial-gradient(
    var(--fill) 1px,
    transparent 0
  );
  background-size: 8px 8px;
}
```

## Text divider

```css
h2 {
  display: flex;
  align-items: center;
  gap: 16px;
}

h2::before,
h2::after {
  content: '';
  height: 1px;
  background-color: #ddd;
  flex-grow: 1;
}
```
